## Project Overview

TyTan is a premium, secure document storage web application designed to provide users with a seamless experience for uploading, managing, and accessing their files. It features user authentication, drag-and-drop file uploads, multiple view modes, advanced search, and storage usage tracking — all wrapped in a modern, responsive design.

---

## Features

- User registration and login with client-side validation
- Drag-and-drop and click-to-upload file functionality
- File management: view, download, and delete files
- Multiple file view modes: grid and list
- Real-time search/filtering of files
- Storage usage tracking with visual progress bar
- Responsive design for desktop and mobile devices
- Animated UI interactions using GSAP and Animate.css
- Secure authentication and session management (client-side)

## Installation and Usage

1. Clone or download the repository to your local machine.

2. Open the project folder.

3. Open the `index.html` file in your preferred web browser.

4. Use the application to register, login, upload, and manage your documents.

> **Note:** This is a client-side application using browser localStorage for data persistence. For production use, integration with a backend and secure storage is recommended.
